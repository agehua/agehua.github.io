解决问题

如果无法查看到datanote，可能有很多问题引起的。
有一个办法，就是重新format namenode，然后再排除其他问题。
具体：删除slave和master的hadoop.tmp.dir内容，然后再次运行hdfs namenode -format。
这种方式，将会清除所有数据。
